Meeting Ingest

Receives a meeting provider's "Transcription complete" webhook and files each meeting into your vault as a #meeting note — automatically.

Set up Fireflies.ai

  1. In Fireflies, create an API key (Settings → Developer).
  2. Write the surface config file at ~/.parachute/surface/state/meeting-ingest.config.json (chmod 600): 
    {
  "fireflies_api_key": "<your Fireflies API key>",
  "fireflies_webhook_secret": "<a long random shared secret>",
  "tag": "meeting"
}
  3. In Fireflies, set the webhook URL to <your-hub-origin>/surface/meeting-ingest/api/webhook/fireflies and the secret to the same value as fireflies_webhook_secret.
  4. Restart the surface (parachute restart surface).

Secrets live only in that config file — they are never stored in a note, logged, or returned by this surface.
