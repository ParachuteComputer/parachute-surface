Meeting MCP

A read-only window onto this vault's meetings. Point an MCP or REST client at the endpoint below to query curated meeting data — recent meetings, full-text search, and a single meeting — without ever touching the raw notes.

MCP endpoint

Stateless Streamable HTTP (POST):

<your-hub-origin>/surface/meeting-mcp/api/mcp

Three tools: recent-meetings, search-meetings, meeting. Each returns curated JSON — domain vocabulary, not raw vault notes.

REST

The same projections are also plain HTTP GETs:

<your-hub-origin>/surface/meeting-mcp/api/recent-meetings?limit=20
<your-hub-origin>/surface/meeting-mcp/api/search-meetings?query=budget
<your-hub-origin>/surface/meeting-mcp/api/meeting?id=<meeting-id>

The shape is the boundary

Every response is the projection's shape of each note — id, title, date, and a summary / snippet / body subset. The raw note (its full transcript, other tags, path, untouched metadata) never leaves this surface. See the package README for how to point this template at a different vault / tag and the public-vs-gated access knob.
